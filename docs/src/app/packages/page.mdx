# Package Distribution

The Native SDK is distributed as a Zig codebase plus a small npm wrapper package for the CLI. The former primitive modules (`geometry`, `assets`, `app_dirs`, `trace`, `app_manifest`, `diagnostics`, and `platform_info`) are now internal Native SDK modules and are available through the main `native_sdk` import instead of standalone Zig packages.

## Install

The npm package is not published yet. Today the CLI builds from a source checkout of the SDK:

```bash
git clone https://github.com/vercel-labs/zero-native.git
cd zero-native
zig build
```

That produces `zig-out/bin/native` — see the [Quick Start](/quick-start) for the full flow. When published, `@native-sdk/cli` will carry prebuilt binaries for macOS (arm64/x64), Linux (gnu/musl, arm64/x64), and Windows (x64); see `packages/native-sdk/` in the repository for the wrapper scripts and packaging metadata.
