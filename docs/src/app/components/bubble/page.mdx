# Bubble

`bubble` is the chat-message surface: a capsule-cornered fill that hugs its message (10px vertical, 12px horizontal — one body line reads as a full pill). Children stack inside, and `variant` picks the side of the conversation: the default muted wash is the received side, `primary` is the sent side — its fill re-inks the text inside automatically, so you never hand-pick knockout ink per message. `outline`, `destructive`, and `ghost` round out the register. Like every stacking surface it rejects `gap`; override the built-in hug with `padding` if you must. Alternate sides with a [spacer](/components/spacer) in a row: spacer-last pushes the bubble left (received), spacer-first pushes it right (sent).

## Markup

```html
<column gap="16" width="340">
  <row>
    <bubble>
      <text wrap="true">Ready to ship the components page?</text>
    </bubble>
    <spacer grow="1" />
  </row>
  <row>
    <spacer grow="1" />
    <bubble variant="primary">
      <text wrap="true">Previews are rendering now.</text>
    </bubble>
  </row>
</column>
```

## Zig builder

```zig
ui.column(.{ .gap = 16, .width = 340 }, .{
    ui.row(.{}, .{
        ui.el(.bubble, .{}, .{
            ui.text(.{ .wrap = true }, "Ready to ship the components page?"),
        }),
        ui.spacer(1),
    }),
    ui.row(.{}, .{
        ui.spacer(1),
        ui.el(.bubble, .{ .variant = .primary }, .{
            ui.text(.{ .wrap = true }, "Previews are rendering now."),
        }),
    }),
})
```
