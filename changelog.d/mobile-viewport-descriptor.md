improvement: **Mobile viewport push takes one descriptor**: `native_sdk_app_viewport` now takes a single `native_sdk_viewport_t` pointer instead of 11 interleaved f32 arguments, so safe-area and keyboard insets survive the self-hosted x86_64 backend (which mis-spilled the later floats, garbling insets on Intel simulators and x86_64 emulators). iOS, Android, and the example hosts build the struct and pass its address.
