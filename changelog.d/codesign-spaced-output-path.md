fix: **Signing a bundle whose path has spaces**: `native package --signing adhoc|identity` now builds the `codesign` invocation as an argv array spawned directly instead of a `sh -c` string, so an output path like `My App.app` stays one argument instead of word-splitting into `My` and `App.app` and leaving the bundle unsigned; this also removes a command-injection surface from paths and identities (#115).
- **Signing failures are loud**: when `codesign` runs and fails (or `--signing identity` is given no `--identity`), `native package` now writes the unsigned state to the signing plan and exits non-zero instead of exiting 0 with a broken bundle — a host that simply cannot sign (cross-packaging a macOS bundle off macOS) still succeeds and records the bundle as unsigned (#115).
