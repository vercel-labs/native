feature: **Opt-in Cairo/Pango packet renderer on Linux**: with `SCHEMIFY_CAIRO=1`, a gpu_surface view draws its canvas packet through Cairo (shapes, gradients, rounded rects, shadows, blur, images) with Pango-shaped text — the Core Graphics + Core Text analog on Linux, mirroring how the macOS host rasterizes on the CPU before the GPU present. Dispatch is payload-structural: any command whose payload the renderer can't paint refuses the frame, so the runtime falls back to the reference renderer — correct, just unaccelerated.
