fix: **No more silent CLI verbs**: `native test` prints the zig build summary (step and test tally) plus a closing `native test: passed` line; `native dev`/`native build` print one-line completions; a failing child step is always named (`native <verb>: zig build step failed (exit code N)`), so a non-zero exit with zero output cannot happen.
- **Help exits 0 everywhere**: `native --help`, `native <verb> --help`, and the `automate`/`markup` subcommand helps all print usage and exit 0; unknown commands and unknown flags print usage and exit 1 — `native init --help` no longer ignores the flag and scaffolds a real app named after the cwd.
