improvement: **App builds default to Debug on the self-hosted backend**: with the Linux host's mixed C-ABI seams now crossed by descriptor pointer, the build graph drops the blanket x86_64 LLVM force. Debug is the default optimize mode (safety checks + markup hot-reload) and uses Zig's self-hosted backend for faster rebuilds; Release still uses LLVM. Consumers select the backend explicitly through `AppOptions`/`MobileLibOptions` `use_llvm`/`use_lld`, and pass `optimize` (or `-Doptimize`) for a release build.
